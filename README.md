# Fling Plugin

A plugin for [Claude Code](https://docs.anthropic.com/en/docs/claude-code) and [pi](https://github.com/badlogic/pi-mono) that adds a skill for setting up and building [Fling](https://flingit.io/) projects — React frontend, Hono API backend, database, storage, cron jobs, secrets, Slack & Discord bots.

## Install

**Claude Code:**

```bash
claude mcp add-skill https://github.com/fling-it/fling
```

**pi:**

```bash
pi install git:github.com/fling-it/fling
```
